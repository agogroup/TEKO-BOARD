## 4. 操作方法

### 4.1 タイムチャート操作

#### 日別ビュー

| 操作 | 動作 |
|------|------|
| アイテム中央をドラッグ | 時間変更 + 別の担当者に移動 |
| アイテム左端をドラッグ | 開始時間を変更（15分スナップ） |
| アイテム右端をドラッグ | 終了時間を変更（15分スナップ） |
| アイテムをタップ | 詳細モーダル表示 |
| 横スクロール | 時間帯を移動 |

#### 週間ビュー

| 操作 | 動作 |
|------|------|
| アイテムをドラッグ | 別の日・別の担当者に移動 |
| アイテムをタップ | 詳細モーダル表示 |

### 4.2 連絡先操作

| 操作 | 動作 |
|------|------|
| [📞] ボタンタップ | 電話アプリ起動（ワンタップ発信） |
| カード全体タップ | 詳細表示 |
